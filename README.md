# React-CRUD

I used this template for saving time to create components, containers, routes, reducer and install several necessary packages.
and this is my CRUD with react project

-------

## Installation

for user prefer npm
```bash
npm install
npm start
```
or user who prefer use yarn
```bash
yarn add
yarn start 
```
